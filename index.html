Privacy Policy – FugeVet_RO Automation

Last updated: 2026-02-06

FugeVet_RO Automation is a helper extension that automates certain steps in the official e-prescription workflow used by veterinarians.

The extension only runs when the user explicitly clicks and triggers the automation. It does not run in the background, does not monitor browsing activity, and does not collect data without user action.

Data Collected

The extension processes only the data required to complete the prescription automation after the user initiates it.

  • Authentication information: used to connect the extension to the user’s Fugevet account.
  • Prescription workflow data: information already entered by the user in the official system.
  • Generated PDF files: prescription PDFs downloaded after user action.
  • System notes: simple tracking information for the user inside the Fugevet system.

No data is collected automatically. All processing happens only after the user clicks the automation button.

Where data is sent

  • https://api.dev1.fugevet.com (Fugevet backend)
  • https://hbsapp.tarbil.gov.tr (official government system)

Data sharing

No user data is sold, rented, or shared with third parties. Data is only transmitted to the Fugevet API to complete the requested automation features.

Data retention

Authentication tokens and temporary data may be stored locally in the browser to maintain the session. Users can remove this data at any time by logging out or uninstalling the extension.

User control

  • The extension only works when manually triggered.
  • No background tracking or monitoring is performed.
  • The extension only operates on the official prescription system pages.
